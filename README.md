# ⚔️ RPG Combat Balance Simulator v1.0
**설계자: 김지훈 (Kim Ji-hoon)**

본 프로젝트는 RPG 게임의 전투 경제 및 성장 구조를 수학적으로 모델링하고 실시간으로 검증하기 위한 시뮬레이터입니다. Excel로 설계된 밸런스 엔진을 Python(Streamlit) 환경으로 이식하여, 유저의 성장 곡선과 전투 효율을 시각화합니다.

## 🚀 시뮬레이터 접속 및 활성화 (중요)
본 앱은 클라우드 프리티어 환경에서 호스팅되고 있습니다. 접속 시 앱이 'Sleep' 상태일 수 있습니다.
- 접속 시 **[Yes, get this app back up]** 버튼이 나타난다면 클릭해 주십시오.
- 시스템이 완전히 가동되기까지 약 **10~30초**가 소요될 수 있습니다.

## 🛠️ 핵심 기능
1. **Planned vs Current 비교**: 기획자가 의도한 표준 성장(CSV Reference)과 유저의 실시간 커스텀 세팅을 비교 분석합니다.
2. **전투 기댓값(E.Dmg) 연산**: 명중률과 치명타 피해(150%~300% 스케일링)가 결합된 실질 화력을 산출합니다.
3. **생존 효율(EHP) 검증**: 방어 상수(K) 기반의 대미지 감소율을 적용하여 실질 체력을 계산합니다.

## 📂 데이터 구조
- `constants.csv`: 방어 상수 K 등 전역 밸런스 설정값
- `player_growth.csv`: 레벨별 표준 성장 데이터 (Reference)
- `monster_db.csv`: 플레이어 화력에 따라 역산된 몬스터 데이터
